# libcarryon

Core engine for the .crn migration format

## Dependencies

| Dependency | Description |
|------------|-------------|
| [tar](https://www.gnu.org/software/tar/) | An utility to manipulate TAR archives |

## Installation

Add the repository URL directly in your `.nimble` file:

```nimscript
requires "https://github.com/NurOS-Linux/libcarryon"
```
