# Contributing to libcarryon

## Getting started

1. Fork the [repository](https://github.com/NurOS-Linux/libcarryon)
2. Clone your fork:
   ```bash
   git clone https://github.com/NurOS-Linux/libcarryon.git
   cd libcarryon
   ```

## Installing for development

```bash
nimble install
```

## Submitting changes

1. Create a branch:
   ```bash
   git checkout -b feat/your-feature
   ```
2. Commit using [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat:` — new feature
   - `fix:` — bug fix
   - `docs:` — documentation only
   - `refactor:` — restructuring without behaviour change
   - `test:` — adding or updating tests
   - `chore:` — maintenance
3. Push and open a Pull Request at https://github.com/NurOS-Linux/libcarryon

## Code style

- Indentation: 4 spaces
- All identifiers in English
- No decorative comment separators (`// ---`, `/* === */`, etc.)

## Reporting issues

Open an issue at https://github.com/NurOS-Linux/libcarryon/issues and include:

- Operating system and version
- Steps to reproduce the problem
- Expected vs actual behavior
- Relevant error output
